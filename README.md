Panoramix
============

Create a panorama using Accelerometer + WebRTC MediaStream APIs of HTML5. It's GNU/GPL.

Test it (requires camera+WebRTC+Accelerometer) : https://app.ecogom.fr/github/nfroidure/Panoramix/master/index.html
